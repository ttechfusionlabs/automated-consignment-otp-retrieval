# Automated Consignment OTP Retrieval System

A production-ready automation system that retrieves consignment OTPs from Outlook email attachments using PowerShell, n8n, and MySQL.

---

## Overview

This project automates the process of extracting OTP information from Outlook email attachments, storing the data in a MySQL database, and making it available for business workflows.

---

## Features

- Automatic Outlook email processing
- OTP extraction from attachments
- MySQL database integration
- PowerShell automation
- n8n workflow integration
- Error logging
- Duplicate prevention
- Production-ready architecture

---

## Technology Stack

- PowerShell
- n8n
- MySQL
- Outlook
- Windows Task Scheduler
